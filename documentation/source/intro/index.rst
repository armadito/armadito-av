Introduction
============

Armadito Antivirus is an open-source antivirus that protects your computers and servers from malwares and viruses. It includes classical signature-based malware detection and provides innovative heuristic detection modules for binaries (MS-Windows and GNU/Linux) and for PDF documents.

An intuitive and user-friendly interface gives access to all Armaditoâ€™s features: on-demand scanning, real-time protection, quarantine zone, threat detection journal, etc. This interface has been developed using state-of-the-art `Node-webkit <http://nwjs.io//>`_ technology. That means that this interface aims to be multi-platform (Windows, Linux, Mac OS X).
