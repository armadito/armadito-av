Licensing
=========

Armadito can be divided in subprojects, which are the following :

- Armadito core
- Armadito gui
- Armadito windows driver
- Armadito module clamav
- Armadito module PDF
- Armadito module H1

**Armadito core**

**Armadito gui**

**Armadito windows driver**
